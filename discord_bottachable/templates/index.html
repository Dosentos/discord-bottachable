About Bottachable

Developed as a school project in Tampere University of Applied Sciences, Finland, Discord Bottachable is an open source bot / web site for easily collecting and viewing desired links from Discord chat.

Links are collected to a database using a bot command and listed on a server specific website. The links can be given tags for searching by topic.

{% else %}

Discord Bottachable is currently in use on {{servers.count}} server(s).

{% endif %}

To add Bottachable to your server you have to have manage server permission on the server.

Instructions:

  1. Click the blue "Add to server"-button

  2. Login to discord on the webpage where you are redirected

  3. Select a server where bot is added from dropdown and click "Authorize"-button

  4. Solve the CAPTCHA if asked

  5. Bottachable should now be on your server
