About Bottachable

Developed as a school project in Tampere University of Applied Sciences, Finland, Discord Bottachable is an open source bot / web site for easily collecting and viewing desired links from Discord chat.

Links are collected to a database using a bot command and listed on a server specific website. The links can be given tags for searching by topic.
